# Odoo Talent Acquisition Go-to-Market Strategy

A premium, consulting-grade static website showcasing a comprehensive Go-to-Market strategy for implementing Odoo in Talent Acquisition operations.

## 📋 Overview

This project presents a complete strategic framework for pitching and implementing Odoo's integrated platform to recruitment agencies and staffing firms. The website features interactive visualizations, detailed market analysis, competitive comparisons, and ready-to-use sales content.

## 🎯 Key Features

### Content Sections
- **Industry Overview** - Market size, growth projections, and business model analysis
- **End-to-End Workflow** - Visual representation of the complete 11-step recruitment process
- **30+ Industry Pain Points** - Categorized challenges with severity indicators
- **Solutions Mapping** - How Odoo addresses each pain point with quantifiable outcomes
- **Odoo Modules** - 12 integrated modules for talent acquisition operations
- **Competitive Comparison** - Matrix comparing Odoo against Bullhorn, Zoho Recruit, Vincere, and others

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No backend dependencies required (static website)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/mailtokarthi12-personal/Talent-Acquisation.git
cd odoo-ta-gtm-website
```

2. Start a local web server:
```bash
# Using Python 3
python3 -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (if installed)
npx http-server
```

3. Open your browser and navigate to:
```
http://localhost:8000
```

## 🔧 Customization

### Modifying Content
Edit the relevant sections in `index.html`:
- Update text content directly in HTML
- Modify section IDs for navigation
- Add/remove sections as needed

### Styling Changes
Edit `styles.css`:
- Modify color variables in `:root` selector
- Adjust typography in `h1-h6` rules
- Update spacing in component classes

### Adding New Sections
1. Add HTML structure in `index.html`
2. Create corresponding CSS in `styles.css`
3. Add JavaScript interactivity in `script.js` if needed
4. Update navigation menu links

## 📱 Responsive Design

The website is fully responsive with breakpoints at:
- **Desktop:** 1200px+ (full layout)
- **Tablet:** 768px-1199px (adjusted grid)
- **Mobile:** Below 768px (single column)
- **Small Mobile:** Below 480px (compact layout)
